# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-02-12

### Added
- Initial release of BAR Unit Sync
- Automatic syncing from GitHub to Webflow CMS
- Support for 11 unit fields (energycost, metalcost, buildtime, etc.)
- Dry-run mode for safe testing
- Auto-publish option for updated items
- Comprehensive error handling and logging
- GitHub Actions workflow for automation
- Detailed documentation and examples
- Test suite for verification

### Features
- Recursive scanning of GitHub units directory
- Intelligent field mapping
- Only updates changed fields
- Detailed sync summary and statistics
- Command-line interface with options

### Documentation
- Complete README with examples
- Contributing guidelines
- MIT License
- Setup script for easy installation
- Test script for verification
